# Kostas Jokes 😂

A Progressive Web App (PWA) that delivers random jokes from multiple joke APIs. Whether you're looking for a quick laugh or just want to explore different types of humor, Kostas Jokes fetches fresh jokes every time you visit.

## Features

- Random jokes fetched from two different joke APIs
- Automatic fallback between APIs for better reliability
- Fully responsive design for desktop and mobile devices
- Progressive Web App (PWA) support
- Installable on supported devices
- Works like a native app when installed
- Fast and lightweight user experience

## How It Works

When a user requests a joke, the application retrieves random jokes from one of two external joke APIs. This ensures a variety of content and improves availability if one service is temporarily unavailable.

## PWA Features

- Installable on mobile and desktop
- App-like experience
- Home screen support
- Offline-ready assets through service workers
- Fast loading and improved performance

## Technologies Used

- HTML
- CSS
- JavaScript
- Service Workers
- Web App Manifest
- External Joke APIs
